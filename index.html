Rhythm Tracks and Contributing

If you'd like to contribute to the rhythm tracks, and know how to use git, you can check out this project at Github - Rhythm Tracks. If you don't know how to use git, you can still download and play with the Garage Band files at the previous link. I like to slow the tempo of the rhythm tracks down for practice and speed them as I get better!
